### DSharpPlusBotについて   
DSharpPlusを使用した、DiscordBotのサンプルです。  
DSharpPlus自体、扱うのが初めてなので（もしかしたら）仕様に慣れないが故の不具合があるかもです。  

### 設定
システム環境変数に以下を追加。

|変数名|値|
|bot_token|DiscordBotのToken|
|---------|-----------------|
|discord_tst_guild_id|DiscordサーバーのID|   

### 機能
- ユーザーの情報表示（埋め込みメッセージのページ形式)
- ヘルプ
- "Hello, World!" の出力
- 入力テスト(string型)
- 入力テスト(DiscordMember)
- ページネーションテスト
- リアクション関連テスト
